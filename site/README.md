# Leen-Co Tech Ltd. Company Website

This is a modern, professional, and responsive company website for Leen-Co Tech Ltd., built with React, Vite, and Tailwind CSS.

## Features
- Homepage with hero section, mission, and CTAs
- About Us (story, values, leadership)
- Services (SaaS products, subdomain links)
- Blog/Insights
- Contact (form, Google Maps)
- Sticky, responsive navbar
- Footer with newsletter, social, and contact info
- Mobile-friendly, SEO-optimized, fast-loading
- Modern color palette: royal/light blue, white, gold
- Modern fonts: Inter, Poppins, Roboto
- Animations (AOS/Framer Motion), testimonials, and more

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.
